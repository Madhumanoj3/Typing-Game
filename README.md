📌 Project Title

#  Typing Trainer & Management System

# 📖 Overview

This project is a full-stack application developed using Java and MongoDB that combines a management system with a typing training platform. It helps beginners improve typing skills while also managing user data, training sessions, and performance records.

The system includes features like user registration, typing lessons, performance tracking, and database management.

# 🚀 Features

👤 User Registration & Login

⌨️ Typing Training Module (finger placement guidance)

📊 Performance Tracking (speed & accuracy)

🧠 Beginner-friendly lessons

💾 Data storage using MongoDB

🔄 CRUD operations (Create, Read, Update, Delete)

💼 Business logic integration (training + management)

# 🛠️ Technologies Used

Frontend: ( JavaFX / CSS )

Backend: Java

Database: MongoDB

Tools: VS Code / Antigravity
